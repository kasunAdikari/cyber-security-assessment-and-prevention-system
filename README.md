# cyber-security-assessment-and-prevention-system
web-based cyber security assessment system using php and NMAP library. special AI validation(scan result) capability available using local AI model with Ollama

to use this proejct you need FPF library, bootstrap library, NMAP, and python(inside the project folder)
also install Ollama and pull llama3 model using "ollama pull llama3" command
install the php compresor 
install xammp
create a database called "cyber"
than insert the .sql query to it
now you good to go
